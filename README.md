# 股票异动雷达

面向 A 股市场的 Android 股票异动监控应用，聚焦“自选股管理 → 行情刷新 → 异动提醒 → 中文语音播报”这条核心链路。

## 核心能力

- 自选股搜索、添加、删除与行情刷新
- 短时波动提醒与台阶提醒
- 中文语音播报与提醒预览
- 提醒历史记录查看
- Android 后台监控与轮询设置
- WebDAV 导入导出
- 面向真机容错的行情刷新策略

## 当前状态

当前仓库提供的是一个可运行、可继续迭代的 Android 原型版本，当前已经覆盖：

- 自选股与提醒规则管理
- 行情刷新与提醒触发
- 提醒历史与设置页
- 中文语音播报链路
- WebDAV 备份与恢复

## 提醒规则

- **短时大幅波动**：在指定分钟窗口内监测涨跌幅变化
- **台阶提醒**：按价格或涨跌幅台阶触发提醒
- 支持针对全部自选股或指定个股配置规则

## 数据来源

- 搜索：东方财富 `suggest` 接口（默认），可切换至新浪搜索
- 行情：东方财富 `push2` 行情接口（默认），支持腾讯、新浪等多源回退
- 多源容错策略已落地：主链路异常时自动切换备用数据源，并配合重试与渐进式刷新降低整页失败概率

> 本项目使用公开数据接口，仅用于学习、研究与产品原型验证。

## 已知限制

- 当前版本主要面向 Android
- 搜索和行情能力依赖第三方公开接口
- 第三方接口限流、变更或不可用时会影响搜索和行情刷新
- 后台监控依赖通知权限、前台服务和电池优化白名单，不同 ROM 表现可能存在差异
- 截图资源仍在逐步补齐，当前以真实截图为主
